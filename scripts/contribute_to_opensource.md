## Quick Start: Contributing to redis-py

### Step 1: Fork and Clone
```bash
# Fork the repository on GitHub first, then:
git clone https://github.com/YOUR_USERNAME/redis-py.git
cd redis-py
```

### Step 2: Create a Branch
```bash
git checkout -b add-examples
```

### Step 3: Add Examples
Create example files showing common redis-py usage patterns.

### Step 4: Commit and Push
```bash
git add .
git commit -m "docs: Add usage examples for common patterns"
git push origin add-examples
```

### Step 5: Create Pull Request
Go to GitHub and create a PR from your branch to the main repository.

## Contribution Tips

1. **Read the CONTRIBUTING.md** file in the repository
2. **Follow the code style** of the project
3. **Write clear commit messages**
4. **Test your changes** before submitting
5. **Be patient** - maintainers are volunteers
